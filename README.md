Ciné'Fahmi
Ciné Fahmi (Cine'Fahmi) is a web platform designed for movie enthusiasts.
It allows users to browse available theaters, select their preferred seats, book tickets, and discover movies similar to those they have already watched.

Features
Theater Booking
View available movie theaters, select seats, and make reservations.

Movie Recommendations
Get personalized suggestions based on movies you have already seen.

Statistics & Analytics
Track popular movies, number of bookings, ratings, and other insights.

Ratings
Rate movies and view aggregated scores from other users.

Seen / Want to Watch
Mark movies you have already watched or plan to see later.

Comments & Reviews
Leave comments and reviews to share your opinion with the community.

Search & Filters
Find movies by genre, rating, or trending status.

Upcoming Releases
Stay updated with upcoming movie releases.

Technologies Used
Backend: PHP (Symfony Framework)
Frontend: HTML, CSS, JavaScript
Database: MySQL
How to Use
Browse available theaters and showtimes.
Select a movie and choose your seats.
Confirm your booking and enjoy the movie.
Rate the movie, leave comments, and mark it as Seen or Want to Watch.
Explore recommended movies based on your viewing history.
Project Goal
Ciné Fahmi aims to provide an interactive and user-friendly movie booking experience while helping users discover new movies through recommendations and community feedback.
